# Chat MVP

Минимальное веб-приложение личного чата. Пользователь может зарегистрироваться, войти, увидеть список других зарегистрированных пользователей и вести личную переписку. Данные пользователей и история сообщений сохраняются в MySQL.

## Технологии

- Node.js
- Express
- EJS
- MySQL
- bcrypt
- express-session
- HTML, CSS, JavaScript

## Возможности

- Регистрация пользователя: имя, фамилия, логин, пароль.
- Авторизация по логину и паролю.
- Список зарегистрированных пользователей после входа.
- Личный чат с выбранным пользователем.
- Сохранение и отображение истории сообщений.
- Простое обновление сообщений без полной перезагрузки страницы.

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Создайте файл `.env` на основе `.env.example` и укажите параметры подключения к MySQL:

```env
PORT=3000
SESSION_SECRET=change_this_secret
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=
DB_NAME=chat_mvp
```

3. Убедитесь, что MySQL запущен и указанный пользователь имеет право создавать базу данных и таблицы.

   Если MySQL не установлен локально, можно поднять базу через Docker:

```bash
docker compose up -d db
```

4. Запустите приложение:

```bash
npm start
```

5. Откройте в браузере:

```text
http://localhost:3000
```

База данных и таблицы `users` и `messages` создаются автоматически при старте приложения.
